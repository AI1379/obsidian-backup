# 简历技术栈八股提纲

这部分我已经拆成三份独立文档，原因很简单：

1. `C++/CUDA` 和 `PyTorch/OpenCV/YOLO`、`TS/JS` 的面试语境差别很大
2. 你对 `C++` 本身比较熟，应该单独给更深一点的准备深度
3. 视觉训练栈和前端语言栈混在一起，不利于速刷

## 文档导航

1. [C++ 与 CUDA 八股提纲](./C++与CUDA八股提纲.md)
2. [Python 与视觉栈八股提纲](./Python与视觉栈八股提纲.md)
3. [TS 与 JS 八股提纲](./TS与JS八股提纲.md)

## 推荐顺序

如果你时间不多，建议这样看：

1. 先刷 `C++ 与 CUDA`
2. 再刷 `Python 与视觉栈`
3. 最后刷 `TS 与 JS`

## 总体防守原则

不管哪一份，统一口径都应该是：

- 你真正熟的点，要能讲到“为什么”
- 你只在项目里用过的点，要把回答收束在工程使用和问题排查
- 不要把自己包装成“所有底层都很熟”

## 你这次和上次的区别

这次拆分后：

- `C++` 部分不再只停在“防守”，而是加了更深入的对象模型、模板、内存模型、异常安全、原子语义等内容
- `Python` 部分补了语言层基础，不再只讲 `Torch/OpenCV/YOLO`
- `TS/JS` 继续保持以高频面试点和项目口径为主
