# 初等数论 · 期末复习参考资料

> 根据课程内容（经典代数数论 + 解析数论入门），精选以下中英文参考书与在线资源。

---

## 📚 核心教材

### 1. Apostol, T. — *Introduction to Analytic Number Theory*
- **出版社**: Springer, UTM (1976)
- **适合度**: ⭐⭐⭐⭐⭐
- **覆盖**: 算术函数 → Dirichlet 卷积 → ζ 函数 → Dirichlet 特征 → L 函数 → Dirichlet 定理完整证明
- **特点**: 本书几乎是本课程解析数论部分的"母本"。前 7 章精确对应课程后半段（4/16 起）的内容。习题丰富，证明清晰。
- **获取**: Z-Library / Library Genesis 有电子版

### 2. 潘承洞、潘承彪 — *初等数论*（第三版）
- **出版社**: 北京大学出版社 (2013)
- **适合度**: ⭐⭐⭐⭐⭐
- **覆盖**: 整除 → 同余 → 二次剩余 → 原根 → 数论函数 → Dirichlet 特征 → 素数分布
- **特点**: 国内最经典的初等数论教材。课程前半段（3/26–4/09）与之高度对应。中文表述详尽，证明步骤清楚。
- **获取**: 浙大图书馆有馆藏

### 3. Ireland & Rosen — *A Classical Introduction to Modern Number Theory*
- **出版社**: Springer, GTM 84 (1990)
- **适合度**: ⭐⭐⭐⭐
- **覆盖**: 二次互反律（多种证明）、Gauss 和、Jacobi 和、ζ 函数、Dirichlet L 函数、分圆域简介
- **特点**: 桥梁式教材，连接初等数论与代数数论。二次互反律的高斯和证明、分圆多项式部分尤其出彩。
- **获取**: Z-Library 有电子版

---

## 📖 辅助读物

### 4. Serre, J.-P. — *A Course in Arithmetic*
- **出版社**: Springer, GTM 7 (1973)
- **适合度**: ⭐⭐⭐⭐
- **覆盖**: 二次型、Hassee-Minkowski 定理、模形式初步
- **特点**: 薄而精（~120页）。第一部分（代数方法）涵盖二次互反律的多种证明、p-adic 数。适合学有余力时拓宽视野。
- **注意**: 风格凝练，跳跃较大，更适合作为"第二遍"读物。

### 5. Davenport, H. — *Multiplicative Number Theory* (3rd ed.)
- **出版社**: Springer, GTM 74 (2000)
- **适合度**: ⭐⭐⭐
- **覆盖**: Dirichlet 级数、L 函数的零点分布、素数定理的解析证明、Siegel 定理
- **特点**: 进阶读物，深入 L 函数理论。如果对 Dirichlet 定理背后更深层的解析数论感兴趣，这是标准参考书。
- **获取**: Z-Library 有电子版

### 6. 柯召、孙琦 — *数论讲义*（上下册）
- **出版社**: 高等教育出版社
- **适合度**: ⭐⭐⭐⭐
- **覆盖**: 经典初等数论全貌，包含大量例题
- **特点**: 老一辈数论学家的经典讲义，习题极其丰富（上下册共数百题），适合刷题巩固。

## 🌐 在线资源

### 7. Terence Tao — *254A: Analytic Number Theory* (UCLA 课程笔记)
- **链接**: https://terrytao.wordpress.com/category/teaching/254a-analytic-number-theory/
- **适合度**: ⭐⭐⭐⭐
- **内容**: Dirichlet 级数、ζ 函数解析延拓、素数定理、Vinogradov 方法
- **特点**: 陶哲轩的讲义，数学严谨又不失直觉。笔记中的"heuristic discussion"对理解"为什么这些方法有效"极有帮助。

### 8. MIT 18.785 — *Analytic Number Theory*
- **链接**: https://ocw.mit.edu/courses/18-785-analytic-number-theory-spring-2023/
- **适合度**: ⭐⭐⭐
- **内容**: 完整的研究生级别解析数论课程（L 函数、自守形式、迹公式等）
- **特点**: MIT OCW 公开课，有视频和作业。内容比本课程深，但前几讲（L 函数、Dirichlet 定理）是很好的补充。

### 9. Keith Conrad — *Expository Papers*
- **链接**: https://kconrad.math.uconn.edu/blurbs/
- **适合度**: ⭐⭐⭐⭐⭐
- **推荐篇目**:
  - *Quadratic reciprocity* — 二次互反律的多种证明
  - *Cyclotomic polynomials* — 分圆多项式的完整处理
  - *Dirichlet's theorem* — Dirichlet 定理的完整证明
  - *Characters of finite abelian groups* — 有限 Abel 群特征理论
- **特点**: Conrad 的 expository notes 可能是互联网上最好的数论补充材料。每篇自成体系，证明极其清晰。

### 10. 3Blue1Brown — *Analytic Continuation of the Riemann Zeta Function*
- **链接**: https://www.youtube.com/watch?v=sD0NjbwqlYw
- **适合度**: ⭐⭐⭐
- **内容**: ζ 函数解析延拓的可视化解释
- **特点**: 如果想对解析延拓建立几何直觉，这 20 分钟的视频比读 20 页书更高效。

## 💡 复习策略建议

1. **优先啃 Apostol 第 6-7 章**：Dirichlet 特征 + L 函数 + 密度定理是课程后半段的核心，Apostol 写得很清楚
2. **二次互反律证明选一种精读**：Eisenstein 的三角证明 vs 高斯和证明，建议深入理解高斯和证明（因为它通向 L 函数）
3. **刷题用 柯召《数论讲义》习题 + 本习题集**：Apostol 的习题也极好
4. **有限 Abel 群 Fourier 分析看 Conrad 的 note**：这部分课程讲得比较快，Conrad 的 note 是绝佳的补充
5. **考前快速过一遍笔记**：7 篇笔记 + 本习题集 = 完整复习闭环
