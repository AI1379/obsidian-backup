# 2026-04-20 复变函数 第3-5节

## 连续介质（Continuum）与拓扑

### 物理背景

张老师从物理力学的"连续介质"概念出发，为 multiple connected domain 做铺垫。

**核心思想**：微观尺度下物质是离散的（$10^{-10}$ m 级别的分子），但宏观尺度（$\sim 0.1$ m）下，通过**积分平均**：

$$\tilde{f}(x, r) = \frac{1}{|B(x,r)|}\int_{B(x,r)} f\,dA$$

可以得到连续函数。这实际上就是广义函数（distribution）的思想——物理测量（如温度计读数）永远是有限半径的积分平均值。

### 尺度的重要性

> "没见过谁吐烟圈吐出一公里的烟圈。" —— Ramree 教授

尺度是决定性的因素。微观与宏观的视角差异，决定了我们看到的是离散还是连续。

### 正则集合（Regular Set）

连续介质的数学刻画：

- **Regular open set**：$(\overline{A^\circ})^\circ = A$（内部的闭包的内部等于自身）
- **Regular closed set**：$\overline{A^\circ} = A$

排除孤立点和"毛刺"——取闭包时线还在，但取内部时线消失了。

### 半解析集（Semi-analytic Set）

正则集合仍有病态情况（如 $\sin(1/x)$ 形边界导致无穷振荡），排除方法：要求边界可由有限个解析函数不等式 $\{g_j \geq 0\}$ 描述。

### Insil（音级）——连续介质的数学模型

**定义**：Insil = regular open + bounded + semi-analytic boundary

三个要素使得 Insil 构成**布尔代数**（支持并、交、补运算）。

### 约当曲线定理与定向

**约当曲线定理**：简单闭曲线将平面分为两个连通分量（有界内部 + 无界外部），曲线本身是它们的公共边界。

**定向的确定**：沿正定向行走时，有界补集始终在**左手边**。

### Insil 的边界表示

**核心定理**：所有 Insil 都可以写成：

$$D = \overline{\text{Int}(\gamma_0)} \setminus \left(\bigcup_j \overline{\text{Int}(\gamma_j)}\right)$$

其中 $\gamma_0$ 是最外层的正定向约当曲线，$\gamma_j$ 是内部洞的负定向约当曲线。

### 哈斯图（Hasse Diagram）与 Betti 数

用包含关系 $\gamma_2 \subset \text{Int}(\gamma_1)$ 定义偏序，形成树形结构（哈斯图）。

- **连通分量数** = 黄色节点（最外层曲线）的个数；
- **洞的个数** = 白色子节点的个数；
- 这些就是**Betti 数**——拓扑不变量。

> 应用：将卡通图像（如熊猫）用 Insil 理论表达——外轮廓 + 眼睛/嘴巴的洞 → 哈斯图数据结构 → 拓扑信息提取。

## Cipher 分解（Cipher Decomposition）

### 自相交曲线的分解

自相交的闭曲线可以被分解为若干个不相交的约当曲线的并集。

**配对算法**：在交叉点处，按顺时针方向将"进入"箭头与最近的"离开"箭头配对。配对完成后，每对连线构成一个约当曲线，且彼此无交叉。

## Disc with n Holes

**定义 3.38**：Connected bounded Insil。外层一圈正定向 + 内部 $n$ 个洞的负定向。

## §3.3.4 柯西积分定理的推广

### 定理 3.71（Disc with n holes 上的柯西定理）

$f$ 在 disc with $n$ holes 上解析 ⟹ 环线积分 = 0（边界正定向）。

**证明思路**：将 disc with $n$ holes 切成 $n+1$ 个 simply connected domain，分别积分后竖线部分正反方向相消。

> **核心思想**：如果 $f$ 在某些点不可导，就把这些点用圆孤立起来（"关禁闭"），外部区域的积分为零。

### 与定理 3.57 的关系

定理 3.71 是 3.57 的推广——令 $n = 0$ 即得 3.57。条件看起来不同，但实质一样。

## §3.4 柯西积分公式

### 定理 3.73（柯西积分公式）

设 $f$ 在包含闭圆盘 $\overline{D(z_0, r)}$ 的区域上解析，则：

$$f(z) = \frac{1}{2\pi i}\oint_\gamma \frac{f(\zeta)}{\zeta - z}\,d\zeta$$

**意义**：解析函数在**内部任意一点**的值，可以用**边界上的环线积分**来表示。

### 关键要素

1. **$2\pi i$**：来自 inversion function $\zeta \mapsto 1/\zeta$ 的环线积分——这是第四章环绕数（winding number）的基础。
2. **被积函数中的 $f(\zeta)$**：不是 $f(z)$ 在积分——$z$ 是固定的内部点。
3. **为什么 inversion 这么重要**？因为 $\frac{1}{\zeta - z}$ 是导数定义中分母的倒数——微分和积分的形式统一。

### 证明结构

1. **引理 3.72**：$\frac{1}{2\pi i}\oint_{|\zeta - z_0|=r} \frac{d\zeta}{\zeta - z} = 1$（$z$ 在圆盘内）——由定理 3.71 + 定向反转得到。
2. 构造 $g(w) = \frac{f(w)}{w - z}$——在 $w = z$ 处有奇点，但利用推论 3.64（star-shaped domain 上扣一点仍有原函数），可将积分变形。
3. 利用柯西积分定理将大圆缩小到任意小的圆——积分与路径无关。

### 推论 3.74（多个洞的情况）

$$\oint_{\gamma_0} \frac{f(\zeta)}{\zeta - z}\,d\zeta = \sum_{j=1}^n \oint_{\gamma_j} \frac{f(\zeta)}{\zeta - z}\,d\zeta$$

每个 $\gamma_j$ 都是正定向的——只要圈之间不交叉、不越界即可。

### 计算柯西积分的"诀窍"

以例 3.75 为例，计算 $\oint_\alpha \frac{z}{z^2 + 1}\,dz$（$\alpha$ 是以原点为心、半径 2 的圆周）：

1. **看圆心与半径**；
2. **找分母零点**：$z^2 + 1 = 0 \Rightarrow z = \pm i$，只有 $i$ 在圆内；
3. **反用柯西积分公式**：将被积函数写成 $\frac{f(z)}{z - z_0}$ 的形式（$f(z) = \frac{z}{z + i}$，$z_0 = i$），直接得到 $2\pi i \cdot f(i) = 2\pi i \cdot \frac{i}{2i} = \pi$。

> "你要算柯西积分的时候，掐着几个诀窍就不会错。" 

### 例 3.76：圆周上的平均值

**Mean Value Property**：解析函数在圆心处的值 = 圆周上的平均值：

$$f(z_0) = \frac{1}{2\pi}\int_0^{2\pi} f(z_0 + re^{i\theta})\,d\theta$$

由柯西积分公式 + 变量代换 $\zeta = z_0 + re^{i\theta}$ 得到。

### 例 3.77：两个零点在圆内

$\oint_\alpha \frac{1}{z^2 + 2}\,dz$：分母零点 $\pm\sqrt{2}i$ 都在圆内，用推论 3.74 拆成两个积分分别计算。

## 定理 3.81：积分号下求导

**Fubini 型定理**：对于含参积分 $F(\alpha) = \int_a^b f(x, \alpha)\,dx$，何时可以 $\frac{d}{d\alpha}F = \int_a^b \frac{\partial f}{\partial \alpha}\,dx$？

**证明难点**：对积分上限求导直接由微积分基本定理得到，但对被积函数求导不能直接交换——需要从 $\varepsilon$-$\delta$ 定义出发，利用一致连续性控制误差。

## 助教习题课

### 习题讲解要点

1. **$z^{2/3}$ 的可微性**：取不同直线 $y = kx$ 逼近原点，极限依赖 $k$，因此不可微。
2. **球面是正则曲面**：用 6 个半球面覆盖，验证参数化的三条性质。
3. **$S^1$ 的覆盖**：证明 $\mathbb{R} \to S^1$ 的指数映射是覆盖映射。
4. **Log 的微分**：用极坐标计算径向和角向的增量变化，得到 Jacobian 矩阵。
