# 透明容器液体残留检测 — 文献调研报告

> **任务背景**：数学建模竞赛题 — 判断透明塑料饮料瓶内是否有残留液体，需构建数据集（≥100 张）、建立检测模型、并尝试扩展至瓶体定位和液体含量分级。
> **技术路线**：YOLO 目标检测 + 传统 CV（边缘检测/Hough 变换等）+ AI 辅助打标
> **调研日期**：2026-05-08

---

## 一、任务拆解与技术需求

| 子任务 | 核心技术 | 可参考方向 |
|--------|---------|-----------|
| 数据集构建 | 自拍摄 + 公开数据 +AI 辅助打标 | Trans10K, LCDTC, SAM 自动标注 |
| 瓶体定位 | YOLO / SSD 目标检测 | 透明物体检测、UAV-BD |
| 液体有无判断 | 区域分析 + 液面检测 + 颜色分割 | 液面识别、液体含量估计 |
| 液体含量分级 | 分类头/回归 | LCDTC 5 级分类、液位估计 |
| 传统 CV 辅助 | Canny/Hough/颜色阈值/纹理分析 | 经典图像处理方法 |

## 二、核心参考文献

### 2.1 透明容器液体检测（直接相关）

#### [A1] Wu et al., "Liquid Content Detection in Transparent Containers: A Benchmark" (Sensors 2023)
- **DOI**: 10.3390/s23156656
- **关键贡献**：提出 LCDTC 数据集（5916 张图片），液体含量分 5 级（empty/little/half/much/full），基线模型 LCD-YOLOF 和 LCD-YOLOX
- **与我们的关系**：⭐⭐⭐⭐⭐ 最直接的参考！任务定义几乎一致。可借鉴：5 级分类框架、Triplet Attention 模块、评估指标设计
- **局限**：场景更多样（不止饮料瓶），数据集中容器种类广泛；我们聚焦饮料瓶可以做得更精
- **引用本文的后续工作**（Semantic Scholar 检索）：
  - "Efficient image feature-driven ML for meniscus volume measurement in glass vials" (2026)
  - "A Real-Time Computer Vision System for Liquid Monitoring in Transparent Containers" (Torres, 2025)
  - "AI-Powered Visual Sensors and Sensing: Where We Are and Where We Are Going" (Nguyen et al., 2025 综述)

#### [A2] Torres, "A Real-Time Computer Vision System for Liquid Monitoring in Transparent Containers" (ACM 2025)
- **URL**: https://dl.acm.org/doi/full/10.1145/3775030.3775041
- **代码**: https://github.com/IsiPaz/LiquidMonitorAI
- **关键贡献**：结合 YOLO 检测 + 传统 CV 色彩分析和分割的**混合方案**
- **与我们的关系**：⭐⭐⭐⭐⭐ 技术路线非常吻合！深度学习目标检测 + 传统图像处理混合方法

### 2.5 传统 CV 方法（Hough 变换/边缘检测/颜色分析）

这些方法虽然不在 arXiv 上大量发表，但在工程实践中非常有效：

#### [E1] Canny 边缘检测 + Hough 直线变换
- **用途**：检测液面水平线（液体在瓶中形成近似水平线）
- **适用场景**：瓶子竖立或近似竖立时，液面是一条清晰的水平线
- **局限**：瓶子横放/倾斜时液面不是直线，需用其他方法

#### [E2] 颜色空间分析（HSV/Lab）
- **用途**：将 ROI 转到 HSV 空间，检测非背景色的液体颜色
- **方法**：对瓶体区域做颜色直方图分析，空瓶 vs 有液瓶在饱和度/亮度分布上有明显差异
- **适用场景**：有色液体（茶、果汁、可乐等）

#### [E3] 纹理分析（LBP/Gabor）
- **用途**：液体区域与空气区域的纹理不同
- **方法**：对瓶体上下区域分别计算 LBP 特征，比较差异

#### [E4] 光流/折射分析
- **用途**：利用液体的折射特性，液体区域和空气区域的光学表现不同
- **适用场景**：视频序列分析

## 三、推荐技术方案

基于文献调研，推荐如下技术路线：

### 阶段一：数据集构建
1. **自拍摄**：收集不同品牌、姿态、液体类型的饮料瓶照片（建议 150-200 张）
2. **公开数据补充**：
   - LCDTC 数据集中的饮料瓶子集
   - Trans10K 中的瓶子类
   - UAV-BD 中的相关图片
3. **AI 辅助打标**：
   - Grounding DINO + SAM 自动生成瓶体 mask
   - 人工修正 + 液体标签标注
   - 工具：LabelMe / CVAT / Roboflow

### 阶段二：核心模型
1. **瓶体定位**：YOLOv8/v11（轻量、成熟、好部署）
2. **液体判断**（推荐混合方案）：
   - **方法 A（纯 DL）**：YOLO 检测瓶体 → 添加液体分类头（类似 LCD-YOLOX 的额外 head）
   - **方法 B（DL+CV 混合，推荐！）**：
     - YOLO 定位瓶体区域 → ROI 裁剪
     - 传统 CV 分析 ROI：
       - Hough 变换检测液面线
       - HSV 颜色分析检测液体颜色区域
       - 上下区域纹理对比
     - 特征融合 → SVM/随机森林/简单 MLP 做最终判断
3. **液体含量分级**：在分类头输出 3 类（少量/中等/较多）或回归

### 阶段三：评估与扩展
- **指标**：Accuracy, Precision, Recall, F1, mAP（如有定位）
- **消融实验**：对比纯 DL vs DL+CV 混合方案
- **泛化测试**：在未见过的瓶型和液体类型上测试
