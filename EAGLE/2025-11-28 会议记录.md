# 项目概述

## 项目名称

"晓曦"

## 背景与目标

随着人工智能技术的快速发展，AI
正在深刻改变人们的生活方式。然而，技术的普惠价值不仅体现在前沿突破，更在于其能否服务于社会中的每一个群体，尤其是视障人士等弱势群体。与此同时，对于刚进入技术领域的实习生而言，如何将理论知识转化为实际工程能力，是成长过程中的关键挑战。

"助盲设备"作为"星程计划"的工程实践项目，旨在通过一个真实、有意义的技术产品开发全过程，帮助实习生从零开始，逐步掌握从调用API、处理数据、构建应用逻辑，到最终进行模型微调（Fine-tuning）的完整 AI 工程能力。项目以"小步快跑、快速迭代"为指导思想，鼓励同学们在实践中学习，在协作中成长。

本项目不仅是一个技术训练项目，更是一次技术向善的探索。我们希望通过开发可部署、可调用的轻量化 AI 服务接口，为未来助盲设备的智能化提供可行的技术原型，同时培养一批具备工程思维、协作能力和社会责任感的年轻开发者。

## 项目目标

### 具体（Specific）

为"星程计划"实习生提供一个贯穿 AI 工程全流程的实践项目，涵盖 API 调用、数据处理、应用开发、模型微调与服务封装等关键环节。

### 可衡量（Measurable）

每位实习生完成至少 3 个递进式开发任务（API 调用 ）

最终交付一个可被调用的 API 接口，并通过基础性能与功能测试。

### 可实现（Achievable）

项目采用"小步快跑"策略，任务模块化、难度阶梯化，配备导师指导与技术文档支持，确保实习生在 8--12 周内可完成一定的阶段性目标。

### 相关（Relevant）

项目内容紧密贴合当前 AI 工程实践趋势，提升实习生在模型部署、服务封装、跨平台调用等方面的实战能力，为其职业发展打下坚实基础

### 有时限（Time-bound）。

项目周期为 2025年9月10日至2026年2月1日，共20周，分三个阶段推进：

第一阶段（第1--5周）：环境搭建与API调用实践

第二阶段（第6--15周）：数据处理与应用原型开发

第三阶段（第15--20周）：模型微调与API服务封装

# 沟通计划 (Communication Plan)

## 沟通频率 

每周1次组会沟通，回顾本周进展、评估风险及规划下周重点。

月度汇报：每月1日，时长30分钟，向管理层提交项目全景报告（进度、风险、资源需求）。

## 沟通方式：会议、邮件、即时通讯、项目看板

通过钉钉视频会议进行周会及月度汇报，确保实时互动与决策。

项目看板：通过钉钉项目看板实时跟踪任务状态（如需求、开发、测试进度），所有成员可自主更新与查看。

## 信息同步对象 

团队内部：同步每日任务进展、技术难点解决方案、代码审查反馈，确保开发协同高效。

实验室老师：定期接收月度汇报、风险评估报告及资源需求建议，支持战略决策。

客户/用户（残联）：项目完成后同步。
