| 模板 | 用途 | Templater |

| [[Course Note]] | 课堂笔记，交互填课程信息 | `tp.date.now` + `tp.system.prompt` |
| [[Math Note]] | 数学笔记，交互填主题 | `tp.date.now` + `tp.system.prompt` |
