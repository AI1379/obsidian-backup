## §3.3.1 代数拓扑基础

### 曲线的连接（Concatenation）

**定义 3.24**：两段曲线首尾相连构成新曲线。注意用词是 **concatenation** 而非 composition（后者有歧义）。

- 两条 smooth curve 的 concatenation 是 piecewise smooth curve。
- 例 3.25：圆弧 + 线段 = piecewise smooth，但曲率不连续（类似早期铁轨的设计缺陷——离心力突变）。
- 正确的曲线过渡应是**欧拉螺线**（Euler spiral），曲率从 0 线性增长。

### 反向曲线（Reciprocal Curve）

**定义 3.26**：将曲线的参数化方向反转（$t \mapsto 1-t$）。

### 路径连通（Path Connected）

**定义 3.27**：集合 $D$ 是 path connected ⟺ 任意两点之间存在 piecewise smooth 曲线连接，且曲线的像在 $D$ 内。

Domain = path connected + open set。

**例 3.28**（非 domain）：$\{z : |z^2 - 3| > 1\}$ 不是 connected 的——正负 $\sqrt{3}$ 之间被虚轴隔开（虚轴上的点不满足条件）。

**例 3.29**（拓扑学家的梳子）：删去 $1/2, 1/4, 1/8, \ldots$ 的区间仍是 path connected 的——可以绕过被删去的点。其证明要点：删去的点的闭包包含 0，而 0 在补集内，所以集合仍是开的。

### Loop（闭路）与同伦（Homotopy）

**定义 3.31**：Loop 是起点与终点相同的连续曲线。

**定义 3.32**：Homotopy（同伦）——两个 path 之间存在连续变形（保持端点固定），用连续映射 $H: [0,1]^2 \to X$ 描述。

### 等价关系的层级

从最严格到最宽松：
1. **全等**（刚性变换：平移 + 旋转）
2. **相似**（放缩 + 全等）
3. **同胚**（Homeomorphism：连续双射）
4. **同伦**（Homotopy：连续变形，比同胚更广义）

### 常微分方程与流映射（Flow Map）

**例 3.33**：常微分方程 $\dot{x} = u(x, t)$（$u$ 关于空间 Lipschitz 连续、关于时间连续）存在唯一解。由此定义流映射 $\varphi(x_0, t_0, t)$——它诱导出 homotopy。

### 引理 3.34（端点固定的同伦是等价关系）

**附加分问题**：若去掉"端点固定"条件，引理是否成立？

**回答**：不成立！原因：连续函数可以将两个点映射到同一点（如常数函数），但反向操作（将一个点"撕开"成两个）不是连续的——**连续函数可以减少连通分量数，但不能增加**。

具体地：流映射一定是双射（因为 ODE 解唯一，系统可逆），但一般的连续函数不一定双射。若端点不固定，自反性不成立——可以把一条线变成环，但不能把环剪开变回线。

### 自由同伦（Free Homotopy）

**定义 3.35**：不固定 base point 的同伦。此时 base point 可以沿 loop 运动，提供了额外的灵活性。

### 球面 vs 环面的拓扑区别

通过 base point 处的 loop 来区分：
- **球面**：任意 simple closed curve 都可缩为一点（只有一个生成元，且平凡）。
- **环面**：存在不能缩为一点的 loop（如经线环和纬线环——两个独立的生成元）。

所有 loop 在 concatenation 运算下构成**群**，球面和环面的群结构不同——这就是**基本群**（Fundamental Group），是拓扑不变量。

### 例 3.37：$S^1$ 上的同伦

$S^1$ 定义为 $[0,1]$ 将首尾粘合。两种典型同伦：
1. **同胚诱导的同伦**：用 $1-t$ 参数联系两个同胚的 loop；
2. **常数同伦**（constant homotopy）：将 loop 缩为一个点。

注意：在 based homotopy 中，base point 在演化过程中可能移动——这正是 free homotopy 提供的灵活性。

### 简单连通（Simply Connected）

**定义**：空间中任意 simple closed curve 都可缩为一点 ⟺ 任意两点间的两条路径同伦。

等价地：在一个 path-connected 的 loop 空间上，选两个点，将 simple closed curve 变成一条线，再缩为点。

## §3.3.2 黎曼证明

### 定理 3.40 的逻辑链

**B ⟹ C**：将 simple closed curve 切成两条路径，其中一条取反向，积分相消。

**C ⟹ A**（构造原函数）：利用引理 2.18 的视角——定义

$$F(z) = \int_{z_0}^z f(\zeta)\,d\zeta$$

然后证明 $F' = f$。为此构造余项 $r(z) = F(z) - F(z_0) - f(z_0)(z - z_0)$，证明 $\frac{|r(z)|}{|z - z_0|} \to 0$。

### 非常数解析函数必有零点（定理 3.45）

**证明思路**：
1. 定义 $Q(z) = z \cdot P(z)$（利用 $1/z$ 的积分表示）；
2. 利用 $1/z$ 的环线积分结果 + 三角不等式；
3. 当 $r \to \infty$ 时，$r^n$ 在分母上导致不等式矛盾。

## §3.3.3 近代证明（Dixon's Proof）

### 不假设导数连续、不要求 simply connected

**定义 3.46**：三角形用**重心坐标**（Barycentric Coordinates）表示——从线性组合变为仿射组合（$t_j \geq 0$，$\sum t_j = 1$）。

### 紧致性（Compactness）

**引理 3.47–3.48**：为定理 3.51 做铺垫。

紧致性的本质是从有限维空间推广时提取出的核心性质：
- **Heine-Borel**：有限维空间中 closed + bounded = compact；
- **无穷维空间**：closed bounded ≠ compact，需要用 sequential compactness 代替。

紧致性有两部分来源：
1. **Closed**：对求极限封闭（有理数的问题在于不封闭）；
2. **Bounded**：有界性。

**Baire 范畴定理**（定理 D.80）：有限个"单薄"集合（如 $\mathbb{Q}$）的并集不可能构成完备度量空间——这是紧致性的**整体刻画**（vs Cantor 交集定理的**局部刻画**）。

> 注：这里更常见的说法似乎是贝尔纲定理，他可以用测度的语言更好的描述。

### 定理 3.51（三角形上的柯西定理）

**证明核心**：

1. 将大三角形四等分（连接三边中点），得到 4 个小三角形；
2. 大三角形上的积分 ≥ 4 × 某个小三角形上的积分；
3. 递归 $n$ 步后：$|\int_{\partial\Delta} f\,dz| \leq 4^n |\int_{\partial\Delta_n} f\,dz|$；
4. 由紧致性（Cantor 交集定理），嵌套三角形序列趋向一点 $z_0$；
5. 由引理 2.18（$f$ 在 $z_0$ 可导），$f(z) = f(z_0) + f'(z_0)(z - z_0) + r(z)$，其中 $|r(z)|/|z - z_0| \to 0$；
6. 前两项的环线积分为零（常数和解析函数），只余余项；
7. $4^n$ 因子与 $2^{-n}$（周长衰减）和 $2^{-n}$（距离衰减）抵消，最终让 $\varepsilon \to 0$。

### 多边形与三角化

**定理 3.52**：将三角形推广到多边形——利用**双耳定理**（Two Ears Theorem）：

> 任意简单多边形至少有两个"耳朵"——可以切下的三角形。

证明思路：将多边形三角化 → 构建对偶图（连接相邻三角形的重心）→ 证明对偶图是树（无环）→ 叶节点就是"耳朵"。

**推论 3.55**：在 simply connected domain 上，$f$ 解析 ⟹ 环线积分为零。

### Star-shaped Domain

**定义**：存在一个点，从该点出发的所有射线都能看到边界。

Star-shaped ⟹ simply connected（但反之不然）。

**定理 3.60**：在 star-shaped domain 上，$f$ 有原函数。

**推论 3.61**（关键推论）：在 star-shaped domain 上扣掉一个点，$f$ 在该点不需要解析，仍有原函数。证明：将含该点的三角形细分，让小三角形的贡献趋向零。

### 重新证明推论 3.15

有了柯西积分定理后，可以用另一种方式理解 $\text{Log}(z)$ 的积分表示——积分路径可以自由选择（如先沿圆弧、再沿径向），分别对应模长（实积分）和辐角（虚积分）的贡献。

## 助教习题课

助教讲解了以下习题：

1. **判断解析性**：利用柯西-黎曼方程，注意方向导数的极限依赖路径选择。
2. **球面是正则曲面**：用 6 个半球面覆盖球面，逐一验证参数化的三条性质（可微、同胚、法向量连续）。
3. **引理 3.9–3.10**：关于积分与极限交换——不能直接定义极限函数（可能不连续），需要利用 $C^1$ 假设和分段估计。
4. **正整数幂映射的连续性**：用几何方法（极坐标下的增量分析）或直接用 $\varepsilon$-$\delta$ 语言证明。
5. **Log 映射的微分**：利用链式法则计算 Jacobian，验证角度变化和长度变化的公式。
