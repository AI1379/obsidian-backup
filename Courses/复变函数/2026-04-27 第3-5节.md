# 2026-04-27 第3-5节 · 复变函数

> 本节概要：从度量空间上的一致收敛过渡到复变函数的解析性质，证明 Weierstrass 定理（局部一致收敛保持解析性），建立 Taylor 展开定理（定理 3.108），并总结解析函数的七条等价刻画（Ultimate Cauchy），最后预告第四章环绕数与辐角原理的思想。

## 本节内容概要

本节是第三章的收尾。张老师首先回顾了附录 D 中度量空间上逐点收敛与一致收敛的理论框架，指出其在求导交换性上的不足，然后引入紧致性作为桥梁（引理 3.99）。在此基础上，利用柯西积分公式的额外结构，证明了复变函数中一致收敛保持解析性（定理 3.101）和级数的正规收敛保持解析性（定理 3.102），最终推导出 Taylor 展开定理（定理 3.108）并给出了若干应用。课程最后对第四章的环绕数、同伦与辐角原理做了思想性的预告。助教陈浩在下半节课讲解了第四、五次作业的重点题目。

## 主要知识点

### 1. 附录 D 复习：逐点收敛 vs 一致收敛

#### 1.1 逐点收敛（Pointwise Convergence）及其局限

逐点收敛的定义 (D.46)：

$$\forall x \in X,\ \forall \varepsilon > 0,\ \exists N = N(\varepsilon, x),\ \text{s.t.}\ \forall n \geq N,\ d(f_n(x), f(x)) < \varepsilon$$

关键特征：$N$ 依赖于 $x$，即 $N = N(\varepsilon, x)$。

逐点收敛的**四大缺陷**（Example D.164–D.167）：
- 两个极限不可交换（$\lim_n \lim_x \neq \lim_x \lim_n$）
- 不保持连续性（$f_n$ 连续但极限 $f$ 可能不连续）
- 不保持极限（极限函数的极限可能不存在）
- 不保持积分与有界性（积分算子与极限算子不可交换）

#### 1.2 一致收敛（Uniform Convergence）

定义 (D.168)：

$$\forall \varepsilon > 0,\ \exists N = N(\varepsilon),\ \text{s.t.}\ \forall n \geq N,\ \forall x \in X,\ d(f_n(x), f(x)) < \varepsilon$$

核心区别：$\exists N$ 与 $\forall x$ 的顺序互换——**全称量词比存在量词强**。

一致收敛的度量刻画：定义函数空间上的度量

$$d_\infty(f, g) = \sup_{x \in X} d_Y(f(x), g(x))$$

使得一致收敛等价于在此度量空间中的收敛。

> **张老师强调**：逐点收敛的定义甚至不要求 $X$ 有拓扑结构（极限在 $Y$ 中取），这本身就暗示该定义"太弱"。

#### 1.3 一致收敛的后果（D.175–D.178）

- **D.175**：极限与求值可交换（$\lim_n f_n(x) = f(x)$ 对所有 $x$ 一致成立）
- **D.178**：若 $f_n$ 连续且一致收敛于 $f$，则 $f$ 连续；且积分与极限可交换

**但求导仍不可交换**（D.179, D.180）：$f_n(x) = \sqrt{x^2 + n^{-2}}$ 一致收敛到 $|x|$，但后者在 $x=0$ 不可导。

D.181（求导交换条件）：需额外假设 $(f_n)$ 在某一点 $x_0$ 收敛，且 $(f_n')$ 一致收敛，才能保证极限与求导可交换。这一"额外条件"源于度量空间本身的结构不足以支撑求导操作。

### 2. 紧致性的桥梁作用（引理 3.99）

**核心思想**：紧致性将"局部一致收敛"提升为"一致收敛"。

设 $K$ 为紧集，$\{f_n\}$ 在 $K$ 上局部一致收敛于 $f$。由紧致性，$K$ 的任意开覆盖有有限子覆盖，即存在有限个小球覆盖 $K$。在每个小球内存在 $N_i$ 使一致收敛条件成立，取 $N = \max_i N_i$，即在整个 $K$ 上获得一致收敛。

> **张老师**：紧致性的定义初看"飘在天上"，但其本质就是为局部性质到全局性质建桥。

### 3. Weierstrass 定理：复变函数中的稳定性（定理 3.101, 3.102）

#### 3.1 定理 3.101（序列版本）

> 设 $(f_n)$ 在开集 $D \subseteq \mathbb{C}$ 上解析，且在 $D$ 上局部一致收敛于 $f$，则：
> 1. $f$ 在 $D$ 上解析；
> 2. $(f_n')$ 在 $D$ 上局部一致收敛于 $f'$。

**与实值函数对比**：定理 3.101 比实分析的对应结论（D.181）少了"在一点收敛"的额外条件——柯西积分公式提供了更强的结构。

**证明思路**：
1. 由 Morera 定理，只需证 $f$ 在任意三角形闭合曲线 $\gamma$ 上的积分为 $0$
2. $\oint_\gamma f\,dz = \lim_n \oint_\gamma f_n\,dz = \lim_n 0 = 0$（由引理 3.99 保证极限可交换）
3. 对 $f_n'$ 用柯西积分公式，逐项求导后用引理 3.99 再次交换极限

> **张老师**：第三章标题从 "Complex Integration" 改为 "Complex Contour Integration"——因为积分在曲线上（紧集！）才使得这些结论成立。

#### 3.2 定理 3.102（级数版本）

> 若级数 $\sum_{n} f_n$ 在 $D$ 上正规收敛（normally convergent）于 $f$，则 $f$ 解析，且 $\sum_n f_n'$ 正规收敛于 $f'$。

正规收敛蕴含局部一致收敛（由 Weierstrass M-判别法），故归结为定理 3.101。

### 4. 定理 3.108：Taylor 展开定理

> 设 $f$ 在开集 $D \subseteq \mathbb{C}$ 上解析，$a \in D$，若圆盘 $\overline{D(a, R)} \subseteq D$，则 $f$ 在该圆盘上有唯一的 Taylor 展开：
>
> $$f(z) = \sum_{n=0}^{\infty} a_n (z - a)^n$$
>
> 其中系数由柯西积分公式给出：
>
> $$a_n = \frac{f^{(n)}(a)}{n!} = \frac{1}{2\pi i} \oint_{|\zeta - a| = \rho} \frac{f(\zeta)}{(\zeta - a)^{n+1}}\,d\zeta, \quad 0 < \rho < R$$

**证明思路**：
1. 对 $|\zeta - a| = \rho$ 上的 $\frac{f(\zeta)}{\zeta - z}$，将 $\frac{1}{\zeta - z}$ 展开为几何级数 $\sum_{n=0}^\infty \frac{(z-a)^n}{(\zeta-a)^{n+1}}$
2. 代入柯西积分公式，交换求和与积分（定理 3.101 保证合法）
3. 逐项积分即得 Taylor 系数

**重要推论——导数的估计**：

$$|f^{(n)}(a)| \leq \frac{n!}{\rho^n} \max_{|\zeta - a| = \rho} |f(\zeta)|$$

> 张老师强调：这意味着解析函数的导数可以被周围的函数值控制——解析函数的行为"特别友好"，不会突然跳到无穷。

### 5. Taylor 展开的若干例子

#### 5.1 例 3.109：$\log z$ 的 Taylor 展开

利用 $\frac{d}{dz}\log z = \frac{1}{z}$（由 $e^{\log z} = z$ 两边求导得到），逐次求导得：

$$\log(1+z) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} z^n, \quad |z| < 1$$

注意：收敛半径为 $1$，可能比从展开中心到定义域边界的距离更大（因定义域扣掉了负实轴）。

#### 5.2 例 3.110：边界上的收敛性

$\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} z^n$ 在 $|z| = 1$ 上除 $z = -1$ 外均收敛，因为 $\log(1+z)$ 仅在 $z = -1$ 时"爆破"。这展示了定理 3.108 提供了具体分析边界收敛性的工具。

#### 5.3 例 3.111–3.112：$\sum \frac{1}{1+z^2}$ 的发散性

实分析中 $\sum_{n=0}^\infty (-1)^n z^{2n}$ 在 $|z| \leq 1, z \neq \pm 1$ 时看似应该收敛（分母无实根），但事实上在 $z = \pm i$ 处发散。复完备性解释了这一现象：复数域中分母的根 $\pm i$ 就在那里。

#### 5.4 例 3.113：$C^\infty$ vs 解析

实值函数中存在 $C^\infty$ 函数没有 Taylor 展开（如标准光滑截断函数），但复变函数中**一次可导即蕴含 Taylor 展开**——这是两个世界的巨大差异。

### 6. Ultimate Cauchy：解析函数的七条等价刻画（§3.4.5）

定理 3.114 总结了解析函数的七种等价描述：

| 编号 | 等价条件 |
|:----:|----------|
| A | $f$ 在 $D$ 上处处复可微（holomorphic） |
| B | $f$ 在 $D$ 上处处可展为收敛幂级数（analytic） |
| C | $f$ 在 $D$ 上连续，且沿任意闭合曲线积分为 $0$（Morera） |
| D | $f$ 满足柯西积分公式 |
| E | $f$ 局部有 Taylor 展开 |
| F | $f$ 的实虚部满足 Cauchy-Riemann 方程 |
| G | $f$ 的实虚部为调和共轭函数 |

> 张老师提到数学界中 "holomorphic" 与 "analytic" 两个学派的用词之争，他倾向于 "analytic"。

### 7. 第四章预告：环绕数与辐角原理

#### 7.1 环绕数的几何意义

$\frac{1}{z}$ 沿闭合曲线的积分 $\frac{1}{2\pi i}\oint_\gamma \frac{dz}{z}$ 给出环绕原点的"圈数"（winding number）。

- 零点在曲线内部 → 积分为 $2\pi i$ 的整数倍
- 零点在外部 → 积分为 $0$

判断点是否在闭合曲线内：从该点出发引一条射线到无穷远，数与曲线的交点个数——奇数在内，偶数在外。

#### 7.2 辐角原理（Argument Principle）

$$\frac{1}{2\pi i}\oint_\gamma \frac{f'(z)}{f(z)}\,dz = Z - P$$

其中 $Z$ 为 $f$ 在 $\gamma$ 内的零点数（计重数），$P$ 为极点数。

> **张老师的比喻**：就像套环游戏——画一个圈，积分一下，看套住了几个根。

#### 7.3 同伦与 Hopf 度数定理

- 同伦：两条闭合曲线之间的连续变形
- Hopf 度数定理将同伦类与映射度联系起来
- 删除平面上不同的点会导致不同的同伦类

## 关键公式与定理

1. **一致收敛度量**：$d_\infty(f, g) = \sup_{x \in X} d_Y(f(x), g(x))$

2. **Weierstrass 定理（3.101）**：局部一致收敛保持解析性，且导数也局部一致收敛

3. **Taylor 展开定理（3.108）**：
$$f(z) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(z-a)^n = \sum_{n=0}^{\infty} \frac{1}{2\pi i}\oint_{|\zeta-a|=\rho} \frac{f(\zeta)}{(\zeta-a)^{n+1}}\,d\zeta \cdot (z-a)^n$$

4. **Cauchy 导数估计**：
$$|f^{(n)}(a)| \leq \frac{n!}{\rho^n} \max_{|\zeta-a|=\rho} |f(\zeta)|$$

5. **$\log z$ 的导数**：$\frac{d}{dz}\log z = \frac{1}{z}$（由 $e^{\log z} = z$ 两边求导）

6. **紧致性桥梁（3.99）**：紧集上的局部一致收敛即一致收敛
