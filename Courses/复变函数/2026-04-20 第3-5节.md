# 2026-04-20 复变函数 第3-5节

## 一、预备知识：连续介质与拓扑

### 1.1 物理背景：从离散到连续

张老师从物理力学的"连续介质"概念出发，为多连通域做铺垫。

- **微观**：物质离散（$10^{-10}$ m 分子尺度）
- **宏观**（$\sim 0.1$ m）：通过**积分平均**得到连续函数

$$\tilde{f}(x, r) = \frac{1}{|B(x,r)|}\int_{B(x,r)} f\,dA$$

这本质上是**广义函数（distribution）**的思想——任何物理测量（如温度计读数）都是有限半径的积分平均值。

> 尺度是决定性的。"没见过谁吐烟圈吐出一公里的烟圈。"——Ramree 教授

### 1.2 正则集合（Regular Set）

连续介质的数学刻画：

| 概念 | 定义 | 直观 |
|------|------|------|
| **Regular open set** | $(\overline{A^\circ})^\circ = A$ | 排除孤立点和"毛刺"——取闭包时线还在，取内部时线消失 |
| **Regular closed set** | $\overline{A^\circ} = A$ | 类似 |

### 1.3 半解析集（Semi-analytic Set）

正则集合仍有病态情况（如 $\sin(1/x)$ 形边界导致无穷振荡）。排除方法：要求边界可由**有限个**解析函数不等式 $\{g_j \geq 0\}$ 描述。

### 1.4 Insil——连续介质的数学模型

**定义**：Insil = **R**egular open + **B**ounded + **S**emi-analytic boundary

三个要素使得 Insil 构成**布尔代数**（封闭于并、交、补运算）。

## 二、约当曲线与 Insil 的边界表示

### 2.1 约当曲线定理（Jordan Curve Theorem）

**定理**：简单闭曲线将平面分为两个连通分量——有界内部 + 无界外部，曲线本身是它们的公共边界。

**定向规则**：沿正定向行走时，有界补集始终在**左手边**。

### 2.2 Insil 的边界表示（核心定理）

所有 Insil 都可以唯一表示为：

$$D = \overline{\text{Int}(\gamma_0)} \setminus \left(\bigcup_j \overline{\text{Int}(\gamma_j)}\right)$$

- $\gamma_0$：最外层**正定向**约当曲线
- $\gamma_j$：内部洞的**负定向**约当曲线

### 2.3 哈斯图（Hasse Diagram）与 Betti 数

用包含关系 $\gamma_2 \subset \text{Int}(\gamma_1)$ 定义偏序，形成**树形结构**（哈斯图）。

- **连通分量数** = 最外层曲线个数（黄色节点）
- **洞的个数** = 内部子节点个数
- 这些即 **Betti 数**——拓扑不变量

> **应用**：卡通图像（如熊猫）→ 外轮廓 + 眼睛/嘴巴的洞 → 哈斯图数据结构 → 拓扑特征提取

### 2.4 Cipher 分解（自相交曲线）

自相交闭曲线可分解为若干不相交约当曲线的并集。

**配对算法**：在交叉点处，按顺时针方向将"进入"箭头与最近的"离开"箭头配对 → 每对连线构成一个约当曲线，彼此无交叉。

### 2.5 Disc with n Holes（定义 3.38）

**定义**：连通的、有界的 Insil。外边界正定向 + 内部 $n$ 个洞负定向。

## 三、§3.3.4 柯西积分定理的推广

### 3.1 定理 3.71（多连通域上的柯西定理）

> 若 $f$ 在 disc with $n$ holes 上解析，则沿整个边界（正定向）的环线积分为零。

**证明思路**：将 disc with $n$ holes 切分成 $n+1$ 个单连通区域，分别积分后切割线上的正反方向相互抵消。

> **核心思想**：$f$ 在奇点不可导 → 把奇点用圆孤立起来（"关禁闭"）→ 外部区域积分为零。

### 3.2 与定理 3.57 的关系

定理 3.71 是 3.57 的推广：令 $n = 0$ 即得定理 3.57。

## 四、§3.4 柯西积分公式

### 4.1 定理 3.73（柯西积分公式）

> 设 $f$ 在包含闭圆盘 $\overline{D(z_0, r)}$ 的区域上解析，则：
>
> $$f(z) = \frac{1}{2\pi i}\oint_\gamma \frac{f(\zeta)}{\zeta - z}\,d\zeta$$

**重大意义**：解析函数在**内部任意一点**的值，完全由**边界上的环线积分**决定。

### 4.2 关键要素解读

| 要素 | 含义 |
|------|------|
| $2\pi i$ | 来自 $\zeta \mapsto 1/\zeta$ 的环线积分——第四章环绕数的基础 |
| $f(\zeta)$ | 不是 $f(z)$——$z$ 是固定的内部点，$\zeta$ 是积分变量 |
| $\frac{1}{\zeta - z}$ | 导数定义中分母的倒数 → 微分与积分的**形式统一** |

### 4.3 证明结构

1. **引理 3.72**：$\frac{1}{2\pi i}\oint_{|\zeta - z_0|=r} \frac{d\zeta}{\zeta - z} = 1$（$z$ 在圆盘内）——由定理 3.71 + 定向反转得到
2. 构造 $g(w) = \frac{f(w)}{w - z}$——$w = z$ 处有奇点，推论 3.64（star-shaped domain 扣一点仍有原函数）允许积分变形
3. 利用柯西积分定理将大圆缩小到任意小的圆 → 积分与路径无关

### 4.4 推论 3.74（多个洞的情况）

$$\oint_{\gamma_0} \frac{f(\zeta)}{\zeta - z}\,d\zeta = \sum_{j=1}^n \oint_{\gamma_j} \frac{f(\zeta)}{\zeta - z}\,d\zeta$$

每个 $\gamma_j$ 均为正定向，圈之间不交叉、不越界即可。

### 4.5 计算技巧与例题

#### 例 3.75：计算 $\oint_\alpha \frac{z}{z^2 + 1}\,dz$（$\alpha$: 以原点为心、半径 2 的圆周）

**解题步骤**：
1. 看圆心与半径
2. 找分母零点：$z^2 + 1 = 0 \Rightarrow z = \pm i$，仅 $i$ 在圆内
3. **反用柯西积分公式**：写成 $\frac{f(z)}{z - z_0}$ 形式
   - $f(z) = \frac{z}{z + i}$，$z_0 = i$
   - $2\pi i \cdot f(i) = 2\pi i \cdot \frac{i}{2i} = \pi$

> "你要算柯西积分的时候，掐着几个诀窍就不会错。"

#### 例 3.76：平均值性质（Mean Value Property）

解析函数在圆心处的值 = 圆周上的平均值：

$$f(z_0) = \frac{1}{2\pi}\int_0^{2\pi} f(z_0 + re^{i\theta})\,d\theta$$

由柯西积分公式 + 变量代换 $\zeta = z_0 + re^{i\theta}$ 得到。

#### 例 3.77：两个零点在圆内

$\oint_\alpha \frac{1}{z^2 + 2}\,dz$：分母零点 $\pm\sqrt{2}i$ 都在圆内 → 用推论 3.74 拆成两个积分分别计算。

## 五、定理 3.81：积分号下求导

**Fubini 型问题**：含参积分 $F(\alpha) = \int_a^b f(x, \alpha)\,dx$，何时可以交换求导与积分？

$$\frac{d}{d\alpha}F = \int_a^b \frac{\partial f}{\partial \alpha}\,dx$$

**证明难点**：
- 对积分上限求导 → 微积分基本定理直接得到
- 对被积函数求导 → **不可直接交换**，需从 $\varepsilon$-$\delta$ 定义出发，用一致连续性控制误差

> 张老师提醒：讲义中有些证明写得简略，不能因为老师说"可以证过去"就跳过——必须自己动手推导。
