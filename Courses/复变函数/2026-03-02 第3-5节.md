# 2026-03-02 第3-5节 · 复变函数

> 本节概要：课程导论课。张老师系统阐述了他的教学理念（少慢精深、思辨精神、知识→力量→热情四层次目标），从"什么是数学"出发，以集合与逻辑为根基，通过自然数的集合论构造（空集→自然数→运算→整数→有理数→实数→复数），一步步展示了数学体系的自下而上构建过程，并引入同构这一核心概念。整节课贯穿"一阴一阳之谓道"的哲学主线。

## 本节内容概要

本节是复变函数课程的第一节课，张老师用了前半节课介绍课程定位、教学理念和评分方式，后半节课正式进入数学内容的构建。

教学理念部分，张老师提出了四层目标：(1) 基本知识、(2) 数学之美、(3) 知识转化为力量（实用能力）、(4) 点燃 passion。他特别强调了"少慢精深"——不照本宣科，而是培养洞察力和思辨精神。评分上采用 40% 作业 + 60% 期末的方式，附加分（extra credit）上不封顶，最终成绩通过仿射变换调分，保证前 10%–20% 满绩。

数学内容部分，张老师从"导数、积分、极限的共同本质是什么"这个问题出发，追溯到 Cauchy 1821 年的微积分著作，引出极限的 ε-δ 定义，再追问"极限是什么"——答案是一个数（number），进而追问这个数属于什么空间。由此展开了一条数系构造链：C ← R ← Q ← Z ← N，最终回到集合与逻辑，并通过 von Neumann 的自然数构造（∅ = 0, S(n) = n ∪ {n}）展示了如何从"无"生出"有"。在此基础上定义了加法（石子合并的抽象），并通过对称性原则诱导出减法，指出数系扩张的内在动力——运算不闭合。

## 一、课程总论与教学理念

### 1.1 为什么这门课与众不同

张老师在 10 年前从美国回来后，发现多数数学课沿用"定义→定理→证明→刷题→考试"的模式，学生感到痛苦。他向学院建议：对于复变函数这样的基础课，应开设 2–3 门平行课，用不同教学方法，把选择权交给学生。

> "如果我就把内容从书上抄到黑板上，大家从黑板上抄下来——那要我干嘛？"

本课的四大特点用四个字概括：**少慢精深**。

### 1.2 四层教学目标

| 层次 | 目标 | 说明 |
|------|------|------|
| 第一层 | 基本知识 | 复变函数的基本概念和定理 |
| 第二层 | 数学之美 | 在知识中感受数学的优美 |
| 第三层 | 知识→力量 | 培养解决实际问题的能力 |
| 第四层 | 点燃 passion | 对复分析产生持久的兴趣和热爱 |

张老师用了一个生动的比喻：

> "假如你碰见一个老神仙，带你练了 4 年降龙十八掌，结果出门被两个小流氓按在地上打一顿——那要么是骗子，要么你练的方式不对。"

数学学好的标志是**洞察力**——面对复杂问题，"看两眼就知道怎么解决"。如果 4 年下来只会说"这题我没刷过"，那就走错路了。

### 1.3 思辨精神

张老师提出了学习的核心原则——**思辨精神**：

- 不要书上写啥就信啥，自己证出来才算数
- 讲义和附录中可能有错误，找到错误（哪怕一个 typo）可获得加分
- 在教学上是不平等的（老师比学生年长），但在做学问上是平等的
- "对就是对，错就是错"

> "荀子曰：学而不思则罔，思而不学则殆。光学没有思，很难把做学问的能力真正培养出来。"

### 1.4 推荐用 LaTeX

张老师强烈建议学生使用 LaTeX：

> "如果你要搞一个特别厚重的东西，那些螺丝不是用手扭的，是要有工具的，而 LaTeX 是我们非常重要的工具。"

他分享了自己的学习方法：在所有课上做"on-the-fly class notes"（随堂笔记），回家后在客厅黑板上假装自己是老师，把当天内容讲一遍，讲得不顺畅就重来。

### 1.5 评分与调分方式

- **比例**：40% 作业 + 60% 期末
- **附加分**：上不封顶。例如 15 次作业每次满分 60 分共 900 分对应 40%，但做完全部附加分的同学可能折算出 50–60 分
- **调分**：张老师采用仿射变换调分——找到自然断点（gap），将前 10%–20% 调至满绩（95），末尾调至 60
- **成绩分布**：历史上的双峰分布——一峰在 90–95，一峰在 60 左右
- **助教**：叶成浩（求是班排名靠前，张老师的博士生）

### 1.6 参考教材与先修课

- **主教材**：Stein 的 *Complex Analysis*（主要讲第 1、2、3、5、8 章）
- **参考书**：一本讲"概念的前生今世"的书（讲概念从哪来、有什么用）
- **先修课**：基础分析（对应附录 B）、线性代数（附录 C，含 Jordan 标准型）

## 二、哲学基础：阴阳与数学

### 2.1 "一阴一阳之谓道"

张老师引用《易经·系辞》：

> "一阴一阳之谓道，百姓日用而不知，仁者见之谓之仁，知者见之谓之知。"

"阴阳"只是一个代号，可以理解为矛盾论中"事物的对立两方面"。阴阳学说对数学的核心启示有三条原则：

| 原则 | 内容 | 数学体现 |
|------|------|----------|
| **万物皆由阴阳所生** | 一切复杂事物都是两种基本要素的排列组合 | 数学 = 集合 + 逻辑 |
| **阴阳对称** | 有阴必有阳，对立面相互定义 | 加法 ⟺ 减法（对称诱导） |
| **阴阳动态转化** | 阴阳不是静态的，而是互相促进、互相转化 | 数系扩张（运算不闭合→扩充数系） |

### 2.2 "糖和脂肪"——数学的本质

张老师讲了一个故事：伯克利国家实验室的同事 No 经营一个小卖部，不管别人拿什么零食来问，他的答案永远是同一个——"Sugar and fat"（糖和脂肪）。

> "你说这学期我们上的是啥？Set and logic. 所有的课就是这个答案，没有其他答案。"

从记事起到现在、到未来，学过的和将要学的所有数学，都只是**集合与逻辑**的排列组合形成的"幻象"。

## 三、数学的源头：从 Cauchy 说起

### 3.1 导数、积分、极限的共同本质

张老师问全班：**导数是什么？积分是什么？这两个问题有一个共同的数学答案——是什么？**

答案是：**极限（limit）**。

- 导数 = 差商的极限
- 积分 = Riemann 和的极限

追溯到 **Cauchy 1821 年**的微积分著作，开篇就问导数和积分是什么，共同答案是极限。

### 3.2 追问极限的本质

张老师用"小孩连环问"的方式不断追问：

- 什么是极限？→ 一个 **number**
- 这个 number 是什么？→ **complex number** ($\mathbb{C}$)
- $\mathbb{C}$ 是什么？→ $\mathbb{R}^2$（实数对的集合）
- $\mathbb{R}$ 是什么？→ 有理数的一种"幻象"（完备化）
- 有理数是什么？→ 整数的一种"幻象"（商域）
- 整数是什么？→ 自然数的一种"幻象"（差集/Grothendieck 群）
- 自然数是什么？→ 集合与逻辑的"幻象"

> "你从 Cauchy 1821 年一直'狗拉羊肠子'（grandmother chaining）到了自然数——但集合和逻辑就是最终的根基。"

## 四、自然数的集合论构造（von Neumann 方案）

### 4.1 Peano 公理

自然数的出发点是 **Peano 公理**（$P_1$–$P_3$），它给出了自然数的条件但不是构造性的。关键条件之一：**数数别数回来了**（不存在 $n$ 使得 $S(n) = 0$），即自然数的拓扑结构是线性的、无环的。

### 4.2 从空集到自然数

张老师在黑板上进行了详细的构造：

**基本符号**：$\emptyset$（空集，"啥也没有"）

**定义**：
- $0 := \emptyset$
- $S(n) := n \cup \{n\}$（successor function，后继函数）

**计算**：

$$0 = \emptyset$$

$$1 = S(0) = 0 \cup \{0\} = \emptyset \cup \{\emptyset\} = \{\emptyset\}$$

$$2 = S(1) = 1 \cup \{1\} = \{\emptyset\} \cup \{\{\emptyset\}\} = \{\emptyset, \{\emptyset\}\}$$

$$3 = S(2) = 2 \cup \{2\} = \{\emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\}\}$$

一般地，$n$ 是一个有 $n$ 个元素的集合，每个元素本身也是一个集合。

> "从 0 开始的那些 1, 2, 3... 都只不过是阴阳互变产生的一个幻象而已。先落到坚实的地面上，先知道我们在干什么。"

### 4.3 加法的定义——从数数到运算

张老师设计了一个课堂互动：**只用石子和数数（数数术），如何计算 $3 + 5 = 8$？**

步骤：
1. 从一堆石子中**数出 3 个**，放到一边
2. 从剩余石子中**数出 5 个**，放到另一边
3. 把两堆石子**合并成一堆**（关键动作！）
4. **重新数**合并后的石子

> "把两堆石子合并成一堆石子——这个动作是有形象的、看得见摸得着的。我们用抽象符号表示它，这就是**加法**。"

关键洞察：**加法 = 合并 + 重新数数**。一旦完成了抽象，就"买一送 $n$"——数石子、数人、数任何东西，都适用。

## 五、从对称性诱导减法 → 数系扩张

### 5.1 对称原则诱导减法

张老师指出：在 $a + b = c$ 中，三个位置 $a$, $b$, $c$ 可以有 6 种排列（三个球放入三个框），但实际上只有两种本质不同的结构：

| 情况 | 形式 | 结果 |
|------|------|------|
| 已知 $a$, $b$，求 $c$ | $a + b = ?$ | **加法**（满足交换律，$a \leftrightarrow b$ 对称） |
| 已知 $a$, $c$，求 $b$ | $a + ? = c$ | **减法**（加法的逆运算） |

> "减法是加法的逆——我们运用阴阳对称的哲学理念，在已有知识基础上找对称，得到新的数学知识。"

### 5.2 运算不闭合 → 数系扩张

减法在自然数上**不闭合**：$3 - 5$ 不是自然数。

> "发现我们的数学体系整个都要崩塌了——怎么办？赶紧补，扩张！"

张老师指出这是阴阳动态转化的体现：
- **数**属阴（被动的，被运算的对象）
- **运算**属阳（动态的，驱使数）
- 发展运算（阳）到一定程度，在原有的数集（阴）上不闭合 → 必须扩充数集（阴适应阳）
- 由此：$\mathbb{N} \xrightarrow{-\, \text{不闭合}} \mathbb{Z} \xrightarrow{\div\, \text{不闭合}} \mathbb{Q} \xrightarrow{\text{Cauchy 完备化}} \mathbb{R} \xrightarrow{x^2 + 1 = 0} \mathbb{C}$

## 六、同构（Isomorphism）

### 6.1 定义

两个代数结构 $(X, f)$ 和 $(Y, g)$ **同构**，记作 $X \cong Y$，如果存在双射 $h: X \to Y$，使得对任意 $a \in X$：

$$h(f(a)) = g(h(a))$$

这可以用交换图（commutative diagram）表示：

```
X ──f──→ X
│h        │h
↓         ↓
Y ──g──→ Y
```

"从这走两步和从那走两步，结果总是一样的。"

### 6.2 与"小无相功"的类比

张老师用《天龙八部》中的鸠摩智作比喻：

> "小无相功的重点是'无相'——无相就是不分别。同构就是在说：这两个结构本质上没有区别，你研究一个就够了。"

同构是"买一送 $n$"的终极版本——世界上有很多不同的问题，如果它们本质一样（同构），就只需解决一个。

## 本节小结

这是一堂非常特别的导论课。张老师几乎没有讲任何复变函数的具体内容，而是：

1. **明确课程定位**：少慢精深、思辨精神、不照本宣科
2. **建立数学的世界观**：数学 = 集合 + 逻辑（"糖和脂肪"）
3. **展示数学的自下而上构建**：从空集出发，通过归纳法构造自然数，通过抽象定义运算，通过对称性诱导新运算，运算不闭合驱动数系扩张
4. **引入核心概念**：同构——数学中"买一送 $n$"的基础设施
5. **贯穿哲学主线**：一阴一阳之谓道——万物由阴阳所生、阴阳对称、阴阳动态转化

整节课的核心信息：**复变函数不只是一门"学公式做题"的课，而是一种思维方式的训练**。脚要先踩在坚实的地面上（集合与逻辑），理解数学体系的根基，才能在上面建高楼。
