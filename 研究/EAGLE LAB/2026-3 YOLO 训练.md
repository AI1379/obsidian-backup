## 1. 交通指示牌数据集

### TT100K (清华 - 腾讯 100K)[1]

| 属性 | 详情 |
|------|------|
| 数据规模 | 100,000 张高分辨率图像（2048×2048 像素） |
| 标注数量 | 30,000+ 个交通标志实例 |
| 类别数 | 221 种交通标志类别 |
| 标注格式 | 边界框、类别标签、像素掩码 |
| 适用任务 | 交通标志检测与分类 |

**主要类别：**
- 限速标志（pl5-pl120, pm5-pm55）
- 禁令标志（p1-p28, pn, pne）
- 警告标志（w1-w66，包括施工、湿滑路面等）
- 高度/宽度限制（ph, pb 系列）
- 指示/指路标志（i1-i15, il, io, ip）

## 2. 路面坑洞与裂缝数据集

### 道路表面缺陷数据集（6000 张）[2]

| 属性 | 详情 |
|------|------|
| 数据规模 | 6,000 张高清图像 |
| 划分比例 | 训练集 70% / 验证集 20% / 测试集 10% |
| 标注格式 | YOLO 格式（txt）+ VOC 格式（xml） |
| 适用框架 | YOLOv5/v8/v9/v10 等 |

**8 大类别：**
1. **Crack** — 道路裂缝
2. **Manhole** — 井盖
3. **Net** — 网状裂缝
4. **Pothole** — 坑洼
5. **Patch-Crack** — 修补裂缝
6. **Patch-Net** — 修补网状裂缝
7. **Patch-Pothole** — 修补坑洼
8. **Other** — 其他异常

### 道路缺陷检测数据集（5321 张）[4]

| 属性 | 详情 |
|------|------|
| 数据规模 | 5,321 张图像 |
| 标注格式 | VOC 格式（XML）+ YOLO 格式（TXT） |
| 总标注框 | 11,104 个 |

**4 大类别：**
- **AlligatorCracking**（龟裂）：1,470 个标注
- **LateralCracking**（横向裂缝）：2,002 个标注
- **LongitudinalCracking**（纵向裂缝）：3,378 个标注
- **Pothole**（坑洞）：4,254 个标注

### 无人机视角道路损害数据集 [5]

| 属性 | 详情 |
|------|------|
| 数据规模 | 2,400 张高清图像 |
| 数据大小 | 2.3GB |
| 拍摄视角 | 无人机航拍视角 |

**6 种损害类型：**
- Potholes (PH) — 坑洞
- Repairs (RP) — 修补
- Oblique Cracks (OC) — 斜裂
- Alligator Cracks (AC) — 鳄鱼裂缝
- Transverse Cracks (TC) — 横向裂缝
- Longitudinal Cracks (LC) — 纵向裂缝
